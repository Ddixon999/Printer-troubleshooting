🖨️ Printer Troubleshooting Guide

Interactive guide for Food & Beverage support team

📋 Level-Setting Phase

Use this checklist to quickly determine whether there is a major issue in the setup of the printer. In order for a seller's printer to work, all of the following must be true:

The printer must be...

  • Compatible with Square POS
  • Loaded with correct paper
  • Powered on
  • Connected correctly
  • Assigned to a Printer Profile

🔍 Discovery Phase

Ask questions to diagnose the seller's issue. Use the process of elimination to rule out potential problems until you find the true root cause.

Key Questions to Ask:

  • Has it worked before or is this your first time setting up?
    If first time, the issue is probably with the setup process. Walk them through setup step-by-step.
  • What is the make and model of the printer you're using?
    Use this info to locate the printer's user manual online.
  • What device is your printer connected to?
    Different Square devices may have different setup procedures.
  • How many printers aren't working?
    Multiple printers = network issue. Single printer = printer/device issue.
  • Are there any lights flashing or error messages being shown?
    The printer's user manual can help diagnose error messages.
  • Is the printer powered on?
    Check wall outlet or power cord if no power.
  • Has anything in your setup recently changed?
    Recent changes may coincide with the issue.
  • Is the printer loaded with the correct paper?
    Incorrect paper may prevent proper function.

🎯 Key Phrases to Listen For:

"Internet is out" / "offline payments" → Ethernet printers may be impacted
"Setting up our new hardware" → Most likely seller error - back to basics
"Just updated our software" → New UI/feature launch causing unexpected behavior
"Recent update to internal network" → Ethernet printers may be impacted
"Error codes" → Refer to user manual to resolve
"Moved/rearranged/relocated devices" → Check all cables are plugged in correctly

🛠️ Solutioning Phase

Take action to resolve the issue based on the type of problem identified.

🟢 Seller Issue

These issues can be resolved through troubleshooting with the seller.

Focus on troubleshooting steps around setup and continue to align on expected behavior.

🔵 Square Issue

Problems with Square hardware or software.

These issues may require an incident to be filed (software) or a warranty (hardware).

🟠 Third-Party Issue

Issues with the seller's ISP or the printer manufacturer.

As the solution falls outside of your scope, you'll provide the best support by referring the seller to the third party directly.

⚡ Quick Fixes

Here are some quick steps you can take which can resolve many printer issues:

First Time Setup: Make sure the printer is set up correctly and a printer profile has been created and assigned appropriately.
Check Hardware Hub: See if the printer is showing as connected.
Power Cycle: Turn the printer off and back on again.
Check Paper: Make sure the printer is not completely out of or low on printer paper.
Cable Check: Ensure all cables are securely plugged into the correct ports. Unplug all cables and replug them in.
Wi-Fi/Ethernet: Turn the device's Wi-Fi off and back on. Double check that both printer and device are on the same network.
Bluetooth: Turn Bluetooth off and back on. Confirm app permissions for bluetooth are enabled (when applicable).
Printer Profile: Make sure there is a printer profile created. If not, have the Seller create one and assign it to the printer!
Device Restart: Turn the device on and off again.

🔧 Printer Setup Guide

Walk sellers through the process of setting up their printer for the first time.

📚 Manufacturer Resources:

  • Epson Support
  • Star Micronics Support
  • Zebra Support

🔌 USB Printer Setup

  1. Connect USB cable and USB adapter (if necessary)
  2. Connect the USB cable to the USB-B port of the printer
  3. Wire the USB cable through cable outlets
  4. Connect the other end to the Square device
  5. Connect power cable and insert into electrical outlet
Note: To transfer data, connect to the USB-B port. If connected to USB-A port, the device will only be charged.

📡 Wi-Fi Printer Setup

  1. Connect power cable and route through cable outlets
  2. Insert power cable into electrical outlet
  3. Connect to Wi-Fi network (refer to printer manual for specific instructions)

🌐 Ethernet Printer Setup

  1. Connect Ethernet cable (category 5 or higher) to router or wall jack
  2. Connect other side to LAN connector of printer
  3. Route the Ethernet cable through cable outlets
  4. Connect power cable and route through outlets
  5. Insert power cable into electrical outlet

📱 Bluetooth Printer Setup

  1. Connect power cable and route through outlets
  2. Insert power cable into electrical outlet
  3. From Square device: Settings > Hardware > Printers
  4. Tap Profiles > Create printer profile
  5. Enter a name for your printer profile
  6. Tap Select Printer > Bluetooth printer
  7. Follow on-screen instructions
  8. Perform a print test
  9. Select what you'll use this printer for
  10. Tap Save

📝 Printer Profiles

Printer Profiles are printer settings which are shared across devices and locations.

🔗 Connect to AirPrint Printer

  1. Square POS App > More > Settings > Hardware
  2. Select Connect AirPrint printer
  3. Choose your printer
  4. Tap Save

📲 Connect to Bluetooth Printer

  1. Square POS App > More > Settings > Hardware > Printers
  2. Select Connect to Bluetooth Printer
  3. Tap Select Printer
  4. Turn the printer on
  5. Go to device Settings > Bluetooth > toggle on
  6. Select the printer (PIN: 1234 if prompted)
  7. On printer: turn on Bluetooth and select your device
  8. Tap Pair when prompted
  9. Add printer name and assign profile

🌐 Connect via Static IP Address (iOS only)

  1. Square POS App > More > Settings > Hardware > Printers
  2. Select Connect Printers
  3. Tap Select Printer > Advanced Printer setup
  4. Select printer type
  5. Enter IP address
  6. Tap Done (will search for printer)
  7. Add Printer Name and Save

➕ Create a Printer Profile

From Square Dashboard:

  1. Dashboard > Settings > Devices > Printer Profiles
  2. Select Create a profile
  3. Add Profile name
  4. Toggle items to print (receipts, order tickets, etc.)
  5. Select store locations
  6. Click Save

From Square POS App:

  1. More > Settings > Hardware > Printers
  2. Tap Profiles tab
  3. Tap Create profile
  4. Add Profile name
  5. Toggle items to print
  6. Select store locations
  7. Save (option to connect printer now or later)
Note: Printer profiles are available on POS within a few seconds on all active devices. You must manually assign a connected printer to the profile on each device.

🌐 Networking Troubleshooting

If an ethernet printer issue is not the result of improper setup or configuration, a poor network connection is often to blame.

📖 Glossary of Networking Terms

Bandwidth The maximum data transfer rate of a network or internet connection, typically measured in bits per second (bps) or megabits per second (Mbps).
Ethernet A widely used networking technology for connecting devices in a LAN.
Firewall A security device or software that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
IP Address A unique string of characters that identifies each computer using the Internet Protocol to communicate over a network.
ISP (Internet Service Provider) An organization that provides services related to accessing, using, managing, or participating in the internet.
LAN (Local Area Network) A network that connects devices in a limited geographical area, such as a home, office, or business.
MAC ID Media access control identifier - a 12-digit hexadecimal number assigned to each device connected to the network.
Modem Modulator-demodulator - a device connecting your home to your internet service provider (ISP) through a physical connection.
Router A device that forwards data packets to the appropriate parts of a computer network. Directs data packets to and from your devices.
Wi-Fi A technology that enables wireless local area networking using radio waves to provide high-speed internet and network connections.
Wi-Fi Bands 2.4 GHz band has longer range but more interference. 5 GHz band offers faster speeds and less interference but shorter range.

⚠️ Signs of Network Issues

  • Slow Internet Speed: May indicate network congestion or ISP problems
  • Intermittent Connectivity: Frequent drops or outages suggest network instability
  • High Latency: Delay causing sluggish responsiveness in payments and POS access
  • Hardware Malfunctions: Abnormal blinking lights or overheating devices
  • Weak Wi-Fi Signal: Dead zones causing poor performance or dropped connections
  • Network Congestion: Slowdowns during peak usage times
  • Frequent Disconnections: Devices having difficulty staying connected
